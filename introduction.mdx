## What is Stagehand?

Stagehand is a browser automation framework used to control web browsers with natural language and code. By combining the power of AI with the precision of code, Stagehand makes web automation flexible, maintainable, and actually reliable.

Unlike traditional automation tools that require low-level DOM manipulation or unpredictable AI agents, Stagehand gives you the flexibility to choose when to use natural language and when to use precise code.

## Why Use Stagehand?

Most existing browser automation tools fall into two categories:

1. **Low-level frameworks** like Selenium, Playwright, or Puppeteer that require brittle selectors and manual maintenance
2. **High-level AI agents** that can be unpredictable and unreliable in production

Stagehand bridges this gap by letting developers choose what to write in code vs. natural language, making it the natural choice for browser automations in production.

## Key Features

### Natural Language Actions

Perform browser actions using plain English instructions:

```python
await session.act(input="click the submit button")
await session.observe(instruction="find the comments link")
```

### Structured Data Extraction

Extract data from web pages into typed schemas:

```python
result = await session.extract(
    instruction="extract the top comment",
    schema={
        "type": "object",
        "properties": {
            "commentText": {"type": "string"},
            "author": {"type": "string"},
        },
    },
)
```

### Autonomous Agent Execution

Run multi-step workflows with autonomous agents:

```python
await session.execute(
    execute_options={
        "instruction": "Find the user's profile and extract their website",
        "max_steps": 10,
    },
)
```

### Async-First Design

Built with modern Python async/await patterns:

```python
async with AsyncStagehand() as client:
    session = await client.sessions.start()
    await session.navigate(url="https://example.com")
    await session.end()
```

## Core Methods

Stagehand provides five core methods for browser automation:

<AccordionGroup>
  <Accordion title="navigate()" icon="compass">
    Navigate to a URL and wait for the page to load.
    
    ```python
    await session.navigate(url="https://example.com")
    ```

  </Accordion>
  
  <Accordion title="act()" icon="hand-pointer">
    Perform actions on the page using natural language or action objects.
    

  </Accordion>
  
  <Accordion title="observe()" icon="eye">
    Find elements and possible actions on the page.

  </Accordion>
  
  <Accordion title="execute()" icon="robot">

    ```python
    await session.execute(
        execute_options={
            "instruction": "Complete the checkout process",
            "max_steps": 5,
        }
    )
    ```
  </Accordion>
</AccordionGroup>

## How It Works

Stagehand operates through a managed browser session:

1. **Start a session** with your preferred AI model
2. **Navigate** to target websites
3. **Use AI-powered methods** to interact with pages
4. **Extract data** or perform actions
5. **End the session** when complete

Each session maintains context and can leverage caching for repeated operations.
