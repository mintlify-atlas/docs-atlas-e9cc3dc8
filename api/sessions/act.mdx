---
title: "Act"
description: "Execute browser actions using natural language or Action objects"
---

## Method Signature

```python
client.sessions.act(
    id: str,
    input: str | dict,
    frame_id: str = None,
    options: dict = None,
    stream_response: bool = False,
) -> SessionActResponse | Stream[StreamEvent]
```

## Parameters

<ParamField path="id" type="string" required>
Unique session identifier returned from `sessions.start()`
</ParamField>

<ParamField path="input" type="string | object" required>
Natural language instruction (e.g., "click the submit button") or an Action object returned from `observe()`

<Expandable title="Action object properties">
  <ParamField path="description" type="string" required>
  Human-readable description of the action
  </ParamField>

  <ParamField path="selector" type="string" required>
  CSS selector or XPath for the element
  </ParamField>

  <ParamField path="method" type="string">
  The method to execute (click, fill, etc.)
  </ParamField>

  <ParamField path="arguments" type="array">
  Arguments to pass to the method
  </ParamField>

  <ParamField path="backend_node_id" type="number">
  Backend node ID for the element
  </ParamField>
</Expandable>
</ParamField>

<ParamField path="frame_id" type="string">
Target frame ID for the action. If not specified, acts in the main frame.
</ParamField>

<ParamField path="options" type="object">
Action options.

<Expandable title="Options properties">
  <ParamField path="model" type="object | string">
  Model configuration object or model name string (e.g., `openai/gpt-4o`)
  </ParamField>

  <ParamField path="timeout" type="number">
  Timeout in milliseconds for the action
  </ParamField>

  <ParamField path="variables" type="object">
  Variables to substitute in the action instruction (e.g., `{"username": "john"}`)
  </ParamField>
</Expandable>
</ParamField>

<ParamField path="stream_response" type="boolean">
Whether to stream the response via Server-Sent Events (SSE). When `True`, returns a `Stream[StreamEvent]`.

You must also set `x_stream_response="true"` header parameter.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
Indicates whether the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
<Expandable title="Data properties">
  <ResponseField name="result" type="object" required>
  <Expandable title="Result properties">
    <ResponseField name="success" type="boolean" required>
    Whether the action completed successfully
    </ResponseField>

    <ResponseField name="message" type="string" required>
    Human-readable result message
    </ResponseField>

    <ResponseField name="action_description" type="string" required>
    Description of the action that was performed
    </ResponseField>

    <ResponseField name="actions" type="array" required>
    List of Action objects that were executed
    </ResponseField>
  </Expandable>
  </ResponseField>

  <ResponseField name="action_id" type="string">
  Action ID for tracking
  </ResponseField>
</Expandable>
</ResponseField>

## Examples

### Basic action with natural language

```python
from stagehand import Stagehand

client = Stagehand()
session = client.sessions.start(model_name="anthropic/claude-sonnet-4-6")

client.sessions.navigate(session.data.session_id, url="https://news.ycombinator.com")

# Perform an action using natural language
response = client.sessions.act(
    session.data.session_id,
    input="Click the comments link for the top post",
)

print(f"Action successful: {response.data.result.success}")
print(f"Message: {response.data.result.message}")
```

### Streaming action

```python
# Stream the action progress
stream = client.sessions.act(
    session.data.session_id,
    input="Fill out the login form with username 'demo' and password 'test123'",
    stream_response=True,
    x_stream_response="true",
)

for event in stream:
    print(f"Event type: {event.type}")
    if event.type == "log":
        print(f"Log: {event.data}")
```

### Using Action objects from observe()

```python
# First, observe available actions
observe_response = client.sessions.observe(
    session.data.session_id,
    instruction="find the search button",
)

if observe_response.data.result:
    action = observe_response.data.result[0]
    
    # Execute the observed action
    act_response = client.sessions.act(
        session.data.session_id,
        input=action,
    )
```

### Action with variables

```python
response = client.sessions.act(
    session.data.session_id,
    input="Type {{search_term}} into the search box",
    options={
        "variables": {
            "search_term": "AI browser automation"
        },
        "timeout": 15000,
    },
)
```

### Custom model for action

```python
response = client.sessions.act(
    session.data.session_id,
    input="Click the download button",
    options={
        "model": "openai/gpt-4o",
    },
)
```

### Async action

```python
import asyncio
from stagehand import AsyncStagehand

async def main():
    async with AsyncStagehand() as client:
        session = await client.sessions.start(model_name="anthropic/claude-sonnet-4-6")
        await client.sessions.navigate(session.data.session_id, url="https://example.com")
        
        response = await client.sessions.act(
            session.data.session_id,
            input="Click the 'Get Started' button",
        )
        
        print(f"Success: {response.data.result.success}")

asyncio.run(main())
```

### Handling action failures

```python
try:
    response = client.sessions.act(
        session.data.session_id,
        input="Click the non-existent button",
        options={"timeout": 5000},
    )
    
    if not response.data.result.success:
        print(f"Action failed: {response.data.result.message}")
except Exception as e:
    print(f"Error: {e}")
```
