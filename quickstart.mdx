This guide will help you run your first browser automation with Stagehand in minutes.

## Prerequisites

<Steps>
  <Step title="Install Stagehand">
    ```bash
    pip install stagehand
    ```
    
    Or with uv (recommended):
    ```bash
    uv pip install stagehand
    ```
  </Step>

  <Step title="Get API Keys">
    You'll need:
    - **Browserbase API Key** and **Project ID** from [browserbase.com](https://browserbase.com)
    - **Model API Key** from your AI provider (e.g., Anthropic, OpenAI)
  </Step>

  <Step title="Set Environment Variables">
    ```bash
    export BROWSERBASE_API_KEY="your-browserbase-api-key"
    export BROWSERBASE_PROJECT_ID="your-browserbase-project-id"
    export MODEL_API_KEY="your-model-api-key"
    ```
  </Step>
</Steps>

## Your First Automation

Let's create a simple script that navigates to a website and performs an action:

```python
import asyncio
import os
from stagehand import AsyncStagehand

async def main():
    # Create the Stagehand client
    async with AsyncStagehand(
        browserbase_api_key=os.environ["BROWSERBASE_API_KEY"],
        browserbase_project_id=os.environ["BROWSERBASE_PROJECT_ID"],
        model_api_key=os.environ["MODEL_API_KEY"],
    ) as client:
        # Start a browser session
        session = await client.sessions.start(
            model_name="anthropic/claude-sonnet-4-6"
        )
        
        print(f"Session started: {session.id}")
        
        # Navigate to a website
        await session.navigate(url="https://www.example.com")
        print("Navigated to example.com")
        
        # Perform an action with natural language
        result = await session.act(
            input="click the 'More information' link"
        )
        
        print(f"Action completed: {result.data.get('success')}")
        
        # End the session
        await session.end()
        print("Session ended")

if __name__ == "__main__":
    asyncio.run(main())
```

<Note>
The `async with` statement ensures the client is properly closed when you're done. You can also use the synchronous `Stagehand` client if you prefer.
</Note>

## Understanding the Code

Let's break down what's happening:

1. **Create Client**: `AsyncStagehand()` initializes the client with your API credentials
2. **Start Session**: `sessions.start()` launches a browser session with your chosen AI model
3. **Navigate**: `session.navigate()` loads a URL in the browser
4. **Act**: `session.act()` performs actions using natural language instructions
5. **End Session**: `session.end()` cleans up the browser session

## Interactive Example

Here's a more interactive example that observes, acts, and extracts data:

```python
import asyncio
import os
from stagehand import AsyncStagehand

async def main():
    async with AsyncStagehand(
        browserbase_api_key=os.environ["BROWSERBASE_API_KEY"],
        browserbase_project_id=os.environ["BROWSERBASE_PROJECT_ID"],
        model_api_key=os.environ["MODEL_API_KEY"],
    ) as client:
        session = await client.sessions.start(
            model_name="anthropic/claude-sonnet-4-6"
        )
        
        # Navigate to Hacker News
        await session.navigate(url="https://news.ycombinator.com")
        
        # Observe available actions
        observations = await session.observe(
            instruction="find the link to the top post"
        )
        print(f"Found {len(observations.data)} possible actions")
        
        # Click the top post
        await session.act(
            input="click the link to the top post"
        )
        
        # Extract structured data
        article = await session.extract(
            instruction="extract the article title and first paragraph",
            schema={
                "type": "object",
                "properties": {
                    "title": {"type": "string"},
                    "content": {"type": "string"}
                },
                "required": ["title"]
            }
        )
        
        print(f"Article: {article.data}")
        
        await session.end()

if __name__ == "__main__":
    asyncio.run(main())
```

## Streaming Example

Get real-time feedback as your automation runs:

```python
import asyncio
import os
from stagehand import AsyncStagehand

async def main():
    async with AsyncStagehand(
        browserbase_api_key=os.environ["BROWSERBASE_API_KEY"],
        browserbase_project_id=os.environ["BROWSERBASE_PROJECT_ID"],
        model_api_key=os.environ["MODEL_API_KEY"],
    ) as client:
        session = await client.sessions.start(
            model_name="anthropic/claude-sonnet-4-6"
        )
        
        await session.navigate(url="https://www.example.com")
        
        # Stream events as the action happens
        stream = await session.act(
            input="click the 'More information' link",
            stream_response=True,
            x_stream_response="true"
        )
        
        async for event in stream:
            if event.type == "log":
                print(f"[LOG] {event.data.message}")
            elif event.type == "system":
                print(f"[SYSTEM] Status: {event.data.status}")
        
        await session.end()

if __name__ == "__main__":
    asyncio.run(main())
```

<Tip>
Streaming is useful for long-running operations. Set both `stream_response=True` and `x_stream_response="true"` to enable it.
</Tip>

## Autonomous Agent Example

Let Stagehand run multi-step workflows autonomously:

```python
import asyncio
import os
from stagehand import AsyncStagehand

async def main():
    async with AsyncStagehand(
        browserbase_api_key=os.environ["BROWSERBASE_API_KEY"],
        browserbase_project_id=os.environ["BROWSERBASE_PROJECT_ID"],
        model_api_key=os.environ["MODEL_API_KEY"],
    ) as client:
        session = await client.sessions.start(
            model_name="anthropic/claude-sonnet-4-6"
        )
        
        await session.navigate(url="https://news.ycombinator.com")
        
        # Let the agent figure out the steps
        result = await session.execute(
            execute_options={
                "instruction": "Find the top post, click it, and summarize the content",
                "max_steps": 5
            }
        )
        
        print(f"Agent completed {len(result.data.actions_taken)} steps")
        print(f"Result: {result.data.result}")
        
        await session.end()

if __name__ == "__main__":
    asyncio.run(main())
```

## Error Handling

Always handle errors in production:

```python
import asyncio
import stagehand
from stagehand import AsyncStagehand

async def main():
    try:
        async with AsyncStagehand() as client:
            session = await client.sessions.start(
                model_name="anthropic/claude-sonnet-4-6"
            )
            
            await session.navigate(url="https://example.com")
            
            result = await session.act(input="click something")
            
            await session.end()
            
    except stagehand.AuthenticationError:
        print("Invalid API credentials")
    except stagehand.RateLimitError:
        print("Rate limit exceeded, try again later")
    except stagehand.APIError as e:
        print(f"API error: {e}")

if __name__ == "__main__":
    asyncio.run(main())
```

## Best Practices

<Accordion title="Use context managers">
  Always use `async with` or `with` to ensure resources are cleaned up properly:
  
  ```python
  async with AsyncStagehand() as client:
      # Your code here
      pass
  # Client is automatically closed
  ```
</Accordion>

<Accordion title="Set appropriate timeouts">
  Configure timeouts based on your use case:
  
  ```python
  client = AsyncStagehand(timeout=30.0)  # 30 seconds
  ```
</Accordion>

<Accordion title="Handle rate limits">
  Implement retry logic for rate limit errors:
  
  ```python
  from tenacity import retry, stop_after_attempt, wait_exponential
  
  @retry(stop=stop_after_attempt(3), wait=wait_exponential(min=1, max=60))
  async def my_automation():
      # Your automation code
      pass
  ```
</Accordion>

<Accordion title="Use streaming for long operations">
  Enable streaming for operations that take more than a few seconds to get real-time feedback.
</Accordion>

<Accordion title="Choose the right model">
  Use faster models like `claude-sonnet-4-6` for most tasks. Use `claude-opus-4-6` for complex reasoning.
</Accordion>

<Accordion title="Minimize AI calls with observe + act">
  Use `observe()` once to find elements, then reuse those Action objects with `act()` to save API calls.
</Accordion>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Authentication Error">
    Verify your API keys are set correctly:
    ```bash
    echo $BROWSERBASE_API_KEY
    echo $MODEL_API_KEY
    ```
  </Accordion>

  <Accordion title="Session Creation Fails">
    Check that your Browserbase project ID is correct and your account has available credits.
  </Accordion>

  <Accordion title="Action Doesn't Work">
    Try being more specific in your instructions:
    - ❌ "click button"
    - ✅ "click the blue Submit button at the bottom of the form"
  </Accordion>

  <Accordion title="Timeout Errors">
    Increase the timeout for slow operations:
    ```python
    client = AsyncStagehand(timeout=120.0)  # 2 minutes
    ```
  </Accordion>
</AccordionGroup>
